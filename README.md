# FUTURE_DS_01
# Online Retail Sales Dashboard By Using Excel

## Project Overview
This project analyzes online retail sales data using Microsoft Excel to identify sales trends, top-selling products, and country-wise performance. An interactive dashboard was created using Pivot Tables, Pivot Charts, and slicers.

## Dataset
https://github.com/Vamshirao07/FUTURE_DS_01/blob/main/online_retail2.xlsb
The dataset contains online retail transaction data including:
Invoice Number
Stock Code
Product Description
Quantity
Invoice Date
Unit Price
Customer ID
Country

 ## Data Cleaning Steps
Removed duplicate records
Handled missing values
Converted date formats
Created Total Sales column
Created Month and Year columns
Converted dataset into Excel table format

## Analysis Performed
Monthly Sales Trend Analysis
Top 10 Selling Products
Country-wise Sales Distribution
KPI Metrics (Total Revenue, Orders, Customers)

1.How do sales change over time?
Sales peak observed during year-end months.
2.Which products generate the most revenue?
Top-selling products contribute major revenue.
3.Which categories or regions are most profitable?
United Kingdom dominates total sales.
4.Where should the business focus to grow faster?
Business growth should focus on high-revunue products and expanding strong-performing regions.

The Online Retail Sales Analysis project demonstrates how raw sales data can be transformed into meaningful business insights using Excel.
